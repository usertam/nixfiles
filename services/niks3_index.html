Using this cache

Nix command Ad-hoc, need trusted-users

nix build nixpkgs#hello \
  --extra-substituters '{{ .CacheURL }}' \
  --extra-trusted-public-keys '{{ .PublicKeysJoined }}'

Nix configuration /etc/nix/nix.conf

extra-substituters = {{ .CacheURL }}
extra-trusted-public-keys = {{ .PublicKeysJoined }}
